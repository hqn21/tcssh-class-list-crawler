# tcssh-class-list-spider
您可以透過此專案爬取臺中二中的上課課表。

## 前置套件

### ``bs4``

## 使用教學

### 1. 下載主要檔案
請下載此專案中的 ``classList.py`` 檔案。
### 2. 打開終端機(Terminal)
### 3. 輸入下方指令
```
python3 
```
※ 請記得指令後方要加上一個空格
### 4. 將下載好的檔案拖入終端機(Terminal)
將下載好的 ``classList.py`` 拖入終端機(Terminal)。
### 5. 根據終端機中出現的訊息設置查詢的班級

## 完整指令
完整指令會如下方所示
```
python3 <classList.py 檔案位置>
```
